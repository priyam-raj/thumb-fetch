# Thumb Fetch

Web-app to fetch YouTube thumbnails without any API.


&nbsp;&nbsp;


## Features

- Automated Light/Dark Mode
- Simple One-Page Navigation
- Fast Results
- No Ads or Paywall, Completely Free & Open-Source
- No API Required
